# Temp_Mail – Temporary Email

**Live Demo** → [https://tamim404.github.io/Temp_Mail/](https://tamim404.github.io/Temp_Mail/)

Instant, beautiful, disposable email. No signup. Self-destructs in 10 minutes.

### Features
- Instant temporary email (mail.tm API)
- 10-minute auto-expire timer (+ extend option)
- Cyberpunk neon + glassmorphism UI
- Matrix rain + interactive particles background
- Dark / Light mode toggle
- One-click copy with haptic feedback
- Auto-refresh inbox
- Toast notifications
- Fully responsive (mobile-ready)
- Zero backend – pure HTML + JS

### Tech Stack
- Tailwind CSS
- Vanilla JavaScript
- Particles.js + Canvas Matrix effect
- Mail.tm free API
